Nataly helps with Objects

The Object is the building block of all JavaScript. The object prototype is how JavaScript deals with inheritance.

Object Properties

Objects have 3 types of properties.

  1. Own properties - these are the properties you will be most familiar with. They are created directly on the object.
  2. Prototype properties - these properties are found through the prototype chain. If JavaScript can't find a property in your own properties, it will search up the prototype chain.
  3. Property Accessors - These are defined with the setter/getter functions.

Functional Instantiation

Functional Instantiation is simply creating objects with functions. You create objects via a function that becomes a 'factory' for creating instances of objects.

Constructor Function

Constructor - is a function, a special function, that is used to create and initialize objets of a particular type. When you create an object using a constructor, it inherits properties and methods defined within that constructor.
