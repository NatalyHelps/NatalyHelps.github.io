Function Definitions

Math.floor()
Rounds a number down to the nearest integer.

Math.random()
Generates a number between 0-1. This includes 0, but does not include 1.

Math.floor(Math.random())
Math.random() randomly returns 0 or a decimal number. Math.floor() then takes this decimal and rounds it down to the nearest integer.
